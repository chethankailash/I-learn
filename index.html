Experiment of its kind, in exploring new technologies for problem solving

Using new machine learning (ML) and mixed reality (MX) technology, the experiment aims at testing the technology and user interaction effectiveness in problem solving.

The approach is built on a system that enables learners to virtually walkthrough and observe 3D visualizations that can be locked and moved in the immersive space.
